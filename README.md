# Expense Tracker Web App

A web-based expense tracker built with Python and Flask.

## Features
- Add expenses with category and description
- View all expenses in a table
- Delete expenses
- Real-time total calculation

## Tech Stack
- Python 3
- Flask
- Jinja2 Templates
- HTML

## How to Run
1. Clone the repository
   git clone https://github.com/swarajpatil10/expense-tracker-web.git
2. Install dependencies
   pip install flask
3. Run the app
   python app.py
4. Open browser at http://127.0.0.1:5000
